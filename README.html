Features

Built for speed, designed to stay out of the way

Every decision in Rook prioritizes the edit. The viewer is always front and center — no panel can obscure it.

⬛
FCP Three-Column Layout
Event library left, viewer center, inspector right. The preview is structurally protected — side panels allocate width from the edges, never from the center.
⚡
Quarter-Res Playback
Drops to quarter resolution during playback for smooth scrubbing on any Apple Silicon chip. Snaps to full resolution the moment you pause.
🎞
FFmpeg Decode
Supports H.264, HEVC, ProRes, and virtually any container FFmpeg can read. No transcoding required to start editing.
🔊
Lock-Free Audio
cpal output through a lock-free ring buffer. Audio and video stay synchronized without blocking the render thread.
📊
VU Meter Bottom Strip
Per-track and master levels live in a dedicated bottom strip. It allocates height — never width — so it can never reduce the preview area.
🎛
Inspector
Per-clip transform, opacity, and speed controls. Clean and non-modal — open when you need it, hidden when you don't.
🗂
Multi-Track Timeline
Blade, lift, and overwrite edits across video and audio tracks. Designed for precision with keyboard-driven workflow.
🌙
Workspace Presets
Default, Audio, Logging, and Color presets snap the workspace to the right tool configuration for the task at hand.
Architecture

Crate structure

Rook is split into focused crates. The UI shell stays thin — media work lives in the engine and decode layers.

rook-core
Data model: Project, Timeline, Track, Clip, Asset. Zero UI dependencies.
rook-decode
FFmpeg probe and decode pipeline. Feeds frames into the compositor.
rook-engine
Edit commands, undo/redo, project mutation. Headless — no UI dependency.
rook-ipc
JSON-RPC agent API. AI agents can inspect, edit, and export projects programmatically.
rook-ui · panels/
preview.rs — video output timeline.rs — multi-track inspector.rs — clip props browser.rs — event library vu_meter.rs — audio levels
Build

From source

Requires Rust 1.78+ and FFmpeg 6+ headers installed via Homebrew.

bash
# Clone and build
git clone https://github.com/xirtus/Rook
cd Rook
cargo build --release -p rook-ui
./target/release/rook-ui
bash — app bundle
cargo build --release -p rook-ui
mkdir -p target/Rook.app/Contents/MacOS
cp target/release/rook-ui target/Rook.app/Contents/MacOS/Rook
codesign --force --deep --sign - target/Rook.app
open target/Rook.app
