## Architecture Quick Reference

### Key Files
```
src/
  main.rs          ‚Äî binary entry, theme setup, mod declarations
  app.rs           ‚Äî RookApp (eframe::App), layout coordinator
  panels/
    inspector.rs   ‚Äî clip properties (collapsible, read-only)
    timeline.rs    ‚Äî tools, tracks, clips, I/O marks, paint, input (~900 lines)
    preview.rs     ‚Äî video canvas, transport, checkerboard fallback
    gallery.rs     ‚Äî asset bin (stub)
    video_preview.rs ‚Äî VideoToolbox decode bridge
  widgets/         ‚Äî stubs (clip_block, playhead, thumbnail, waveform)
```

### Core Crates
- `rook-core` ‚Äî Clip, Track, Timeline, Project, EditCommand, Effect, Transform (data model)
- `rook-engine` ‚Äî Engine (apply commands, undo/redo, save/load, import/export)
- `rook-timeline` ‚Äî TimelineGraph (DAG for export/agents)
- `rook-renderer` ‚Äî GPU compositor (wgpu)

### Borrowing Pattern (timeline.rs)
- Read snapshot data upfront (durations, track count)
- Paint from snapshots (no borrow held)
- Defer mutations (collect IDs, apply after loops)
- `handle_input()` returns early for blade/zoom tools before hitting drag logic

### Clip Splitting
```rust
self.split_clip_at(project, clip_id, at_frame);
// Creates left/right clips, inserts both, selects right
```

### To Add Inspector Editing
1. Change `show(&mut self, ui, project: &mut Project)`  
2. In `app.rs`: call `self.inspector.show(ui, engine.project_mut())`
3. Add `egui::Slider`/`DragValue` widgets for each property
4. Apply via `engine.apply(EditCommand::SetClipTransform { ... })`
